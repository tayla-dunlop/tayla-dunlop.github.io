What Are Classes And Ids?

Classes and Ids Are a tool used in CSS to help style certain parts of your work. Lets say you want make all your headers the color red without having to make all your text on your website red, this is when classes and Ids would become your bestfriend. On this page I will go into slightly more depth about the difference of classes and Ids aswell as when you should either one. So Again, sit back, relax, and enjoy the ride!

How To Use a Class

A class is exactly what you would think it is, a way to class/catagorise things. A class is typically used as a more broad catagory for elements compared to Ids. For example, a class would be used to turn all the p elements font to monospace.

Now, how do you actually create a class? I'm glad you asked! Lets follow a list of steps on how to do it!
  • Make sure you are in your CSS file
  • Decide on a class name (lets just call it diff font)
  • Create the class as shown: .Diff-font (with classes its very important to have the .)
  • Now, after the .diff-font you will put the fancy curly brackets {}
  • Inside the brackets you will chose what it is you want to do (in this case change a font)
  • In the curly brackets enter the following font-family: monospace;
  • After this you will now enter your html file and apply the class to any of your elements!
  • To apply to an element do the following class="diff-font" and away you go!

How To Use an Id

Ids are almost identical to classes, but there are slight differences, especially when it comes to applying them. As mentioned above, classes are used to catagorise more broadly, whereas Ids are used to apply a certain styling rule to one particular element. Although this isnt a strict rule, it is common practice. Now, creating an Id is the exact same as a class except for 1 important detail. This is that instead of using the period (.) at the beginning, you use the hash (#). So now go forth you blossoming bird and create an Id using the list above and make changes as you need! To Summerise

Overall, classes and Ids are very similar but there is a time when each of them should be used. Classes should be used for broader catagorising such as header colors, text sizes, fonts and many more things. Ids should be used for 1 element or one line, for example if you wanted to highlight a single line and make it red an Id would be put to perfect use.
